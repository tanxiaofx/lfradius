# lfradius
## 柳州城市职业学院(LCVC)校园网Openwrt自动认证脚本





> [!WARNING]
> （特别提醒，该脚本使用了AI工具生成，仅在红米ax3000上测试，其他设备兼容性未知，脚本稳定性、副作用未知。如果您对AI工具有反感，请不要下载这个脚本）




一：简介
柳州城职院使用凌风（lfradius）认证计费系统和panabit的全自动身份认证系统，
该校园网的特殊机制——“JS 自动跳转 + cmcc_login_value二次提交”：传统简单的用户名/密码 POST 会被服务器拒绝（返回假成功页面）；
脚本通过抓取服务器返回的隐藏 cmcc_login_value（Base64 编码的 JSON），并进行第二次 POST 提交，从而真正完成上线流程。




二：具体认证流程为：客户端（路由器）
        -
     Portal 页面（HTML）
        -
    第一次 POST（用户名/密码）
        -
    服务器返回 JS 自动提交表单
        -
    第二次 POST（cmcc_login_value）
        -
    ✅ 放行上网



	
三：该脚本的核心目的：脚本的核心目标是绕过人工网页登录，通过命令行模拟浏览器行为，实现路由器（OpenWrt）在连接校园网时的全自动身份认证。



四：安装该脚本的前置条件：



1: 必须使用iptabls的ttl命令修改出口ttl为64`iptables -t mangle -A PREROUTING -j TTL --ttl-set 64`



2: 必须安装curl组件


五：注意事项：


1:sh脚本需要添加上你的学号和密码后再复制粘贴到终端运行，不能直接上传sh文件到root文件夹下


2:该校园网账号所绑定的手机卡不得欠费，套餐过期等原因，否则该脚本不起作用


3:需要在 系统——启动项——本地启动脚本添加  `/root/campus_login.sh ` 实现开机自启
